# 2.2.9

$$f(x) = \frac{1}{1-x}$$である。
$$f^k(x)$$をいくつかの$$k$$について試してみる。

$$f^2(x) = \frac{1}{1-\frac{1}{1-x}} =  \frac{1-x}{1-x-1} = 1 - \frac{1}{x}$$

$$f^3(x) = 1 - (1-x) = x$$

$$f^4(x) = \frac{1}{1-x} = f^1(x)$$

すなわち$$k \geq 4$$のとき$$f^k(x) = f^{k-3}(x)$$なので、
求める値は
$$f^{1999}(2000) = f^{1996}(2000) = \cdots =f^1(x) = \frac{-1}{1999}$$
である。
