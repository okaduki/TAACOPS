# 2.3.32

仮定より$$x+x^{-1}$$は整数である。
任意の自然数$$n\geq 1$$に対して$$x^n+x^{-n}$$が整数であることを累積帰納法で示す。

$$n>1$$のとき$$(x^{n-1} + x^{-(n-1)})(x + x^{-1}) = (x^n + x^{-n}) + (x^{n-2} + x^{-(n-2)})$$
なので$$x^n + x^{-n} = (x^{n-1} + x^{-(n-1)})(x + x^{-1}) + (x^{n-2} + x^{-(n-2)})$$である。
帰納法の仮定より右辺に現れる全ての項は整数であることが言えるので、左辺の$$x^n+x^{-n}$$も整数である。
